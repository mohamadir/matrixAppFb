## Installation
  
### 1. clone the project 
```ruby
git clone git@github.com:mohamadir/matrixAppFb.git
```

### 2. go to project directory
```ruby
cd matrixAppFb
```

### 3. install packages
```ruby
npm install
```
### 4. connect device

connect android device / open an emulator 

### 5. run the app
```ruby
react-native run-android
```
